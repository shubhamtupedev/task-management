# task-management

Task Management System

A Spring Boot REST API application that provides CRUD operations for managing tasks with input validation and clean layered architecture.

Tech Stack:
- Java
- Spring Boot
- Spring Data JPA
- Hibernate Validation
- PostgreSQL

Features:
- Create, update status, delete, and fetch tasks
- Request validation using Bean Validation
- Layered architecture (Controller, Service, Repository)
- Exception handling
- Clean RESTful APIs

Future Enhancements:
- Authentication & authorization
- Pagination and sorting
- Swagger integration
- UI Integration(React JS)
- Unit testing
- Search and filtering capabilities
- Docker support for containerization
- Deploy on AWS
- Logging
